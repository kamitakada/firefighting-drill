# 中野区 初期消火機器操法大会 対策ガイド

中野区町会連合会主催の初期消火機器操法大会に向けた対策・学習用ホームページです。

## 収録種目
- スタンドパイプ操法（指揮者・1番員・係員のアクション時系列）
- バケツリレー（競技ルール・手順・セリフ）

## 動作要領・参考データ
本サイトのコンテンツは、中野区町会連合会の公式情報および動画を参考に作成しています。

- [中野区町会連合会「初期消火機器操法大会 操法基準」](https://nakano-chokai.jp/info/初期消火機器操法大会%E3%80%80操法基準/)
- スタンドパイプ操法 公式動画: [https://youtu.be/4VLNx8_SW3s](https://youtu.be/4VLNx8_SW3s)

## 開発・プレビュー方法
ローカル環境でプレビューを動かす場合は以下を実行してください。

```bash
npm install
npm run dev
```
